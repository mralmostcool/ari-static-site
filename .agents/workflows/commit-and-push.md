// turbo-all
1. Run `git status` to see what changed.
2. Run `git add -A`.
3. Write a clear, conventional-commit-style message summarizing the changes.
4. Run `git commit -m "<message>"`.
5. Run `git push` (use `git push -u origin <branch-name>` if no upstream is set).
6. If nothing has changed, report that instead of committing an empty commit.
